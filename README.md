# Hospital Management System (Java)

## Overview
The **Hospital Management System** is a Java-based console application designed to streamline hospital operations. It allows users to manage **patients, doctors, and appointments** efficiently. This project demonstrates **object-oriented programming (OOP)**, **database connectivity (MySQL)**, and **basic hospital management operations**, making it a portfolio-ready project for learning and showcasing Java skills.

---

## Features
- **Patient Management**: Add, update, delete and view patient records  
- **Doctor Management**: Add, update, delete and view doctor records  
- **Appointment Management**: Book, view and delete appointments  
- **Database Integration**: Persistent data storage with MySQL

---

## Technologies Used
- **Java SE** – Core application development  
- **MySQL** – Database backend  
- **JDBC** – Java Database Connectivity  
- **Git & GitHub** – Version control
